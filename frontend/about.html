ABOUT

My Library is a minimal book and quote tracker built with HTML, CSS, and vanilla JavaScript — connected to a Node/Express backend that stores data directly in a GitHub repository.

Each book can have ratings (for users A and N), quotes, and favorite words or phrases. All data updates are automatically committed to books.json in the GitHub repo via the backend.

Librarian System

The Librarian menu (☰ icon on the top right) lets you manage your collection:

  • Add multiple books at once (one per line)
  • Remove multiple books by their numbers, with confirmation
  • Rename a single book by number
  • Sort (coming soon)

Files in this project

  • frontend/index.html – main app UI
  • frontend/style.css – visual design
  • frontend/js/*.js – frontend logic (modular)
  • backend/server.js – backend API (on Render)
  • books.json – data file stored in GitHub

Hosted on GitHub Pages · Backend deployed via Render · All changes automatically synced to GitHub
