# My Library

A simple book + quote tracker built with HTML, CSS, and vanilla JS.

- No backend
- Data is saved in `localStorage` in the browser
- Ready for GitHub Pages

## Files

- `index.html` – main UI
- `style.css` – styling
- `script.js` – tabs, books, quotes, localStorage
- `about.html` – info page

## Run locally

Just open `index.html` in a browser.

## Deploy on GitHub Pages

1. Create a new repo and upload these files.
2. Go to **Settings → Pages**.
3. Choose **Deploy from branch**, select `main` and `/ (root)`.
4. Wait for GitHub Pages to give you a URL.

That’s it.
